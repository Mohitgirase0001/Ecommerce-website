Home Goods Accessories Website 🏡🛍️
Welcome to the Home Goods Accessories Website, an e-commerce platform where users can explore and purchase a variety of stylish and high-quality home accessories to enhance their living spaces.

🌟 Features
✅ User-Friendly Interface – Easy navigation and clean design for a smooth shopping experience.
✅ Product Categories – Browse different home accessories, including decor, furniture, kitchenware, and more.
✅ Secure Checkout – Safe and hassle-free payment processing.
✅ Wishlist & Cart – Save favorite items and purchase later.
✅ Responsive Design – Works seamlessly on desktop, tablet, and mobile devices.

🚀 Technologies Used
Frontend: HTML, CSS, JavaScript
Database: MySQL
Version Control: Git & GitHub
